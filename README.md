# Executável para instalar os drivers do MT7922

Notei que algumas placas de rede tem drivers que contém  vírus. Não todas, mas algumas tem e no caso do Windows, a sua ferramenta Windows Update pode demorar para reconhecer a placa de rede e instalar os drivers necessários. Por isso, esse repositório foi criado, para poder instalar os drivers presentes no Catálogo do Windows Update de forma forçada no sistema. Todos os arquivos presentes na pasta `Windows/WiFI` e `Windows/BT` podem ser encontrados em [Catálogo Microsoft Update](https://www.catalog.update.microsoft.com)
